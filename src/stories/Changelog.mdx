# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2025-10-23

### Added

- **AlertDialog**: Radix-based alert dialog with `danger` and `neutral` variants, contextual styling, focus management, and a11y.
- **Modal**: Radix-based dialog with animated overlay, centered content, and `inside` / `outside` scroll behavior.
- **Toast**: Notification system with provider, hook, and variants (`success`, `info`, `warning`, `danger`); supports position, duration, and stacking.

### Documentation

- Added Storybook stories for **AlertDialog**, **Modal**, and **Toast** with usage examples.

### Testing

- Added unit tests for all the components (Vitest + React Testing Library).

## [1.0.1] - 2025-10-03

### Added

- **Documentation**: Created a new polished "Welcome" introduction page with a live component showcase.
- **Documentation**: Added a detailed "Installation" guide.
- **Theming**: Implemented a custom theme for the Storybook UI, including a logo and brand colors.
